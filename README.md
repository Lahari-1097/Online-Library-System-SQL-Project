# Online-Library-System-SQL-Project

## Project Overview

** Project Title **: Online Bookstore Analysis
** Level **: Beginner to Intermediate
** Database **: `OnlineBookstore`

This project is designed to demonstrate SQL skills and techniques typically used by data analysts to design relational databases, import structured data, and perform business-driven analysis. The project involves creating an online bookstore database, importing CSV files, performing exploratory data analysis (EDA), and solving real-world business questions using SQL queries.

The project uses three CSV datasets:

1. **Books**
2. **Customers**
3. **Orders**

This project is ideal for those who want to strengthen their SQL fundamentals including joins, aggregations, filtering, grouping, and data integrity constraints.
